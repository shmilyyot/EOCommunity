# 一个基于ssm的论坛
	主体使用传统的ssm，但集成了更多组件。提供了用户端所有基本功能，暂没有管理员角色的功能。
	主要实现了：用户登陆注册、登陆状态、热门板块、热门帖子、搜索、收藏夹、消息通知、修改资料和头像、发帖、评论、回复等功能。
